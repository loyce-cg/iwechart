Player Helper Window

To get window handler, type in console:
win = privmxViewRequire("Starter").Starter.viewInstance;
Toogle debug log level:
win.voiceChatModule.toogleDebugLogLevel();
Create 2s ping drop
win.voiceChatModule.currentServerApi.sendingQueue.delay(2000);
Set speed up ratio
win.voiceChatModule.currentServerApi.recorder.speedUp = 1.2;
